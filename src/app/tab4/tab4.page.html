Privacy & Security

🔐 Welcome to the Privacy & Security Center

Your trust matters to us. This page outlines how your inventory data is handled, protected, and respected.

Key Policies
  • 🔒 Data Security: All data operations are secured with HTTPS protocols.
  • 👤 User Privacy: We don’t store personal identifiers. No login info is required.
  • 🧹 Data Access: Only data you create or edit is visible to you.
  • ♻️ Updates: Every update overwrites previous values — there’s no versioning.
  • 🗑️ Deletion: Items can be deleted, except protected records like "Laptop".
  • 🧾 Data Logs: No activity logs are stored or shared — your actions remain private.
  • 📦 Storage: The backend database is hosted securely with institutional oversight.
